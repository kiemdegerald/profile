Mes Technologies Préférées

Django

Framework Python puissant que j'utilise pour développer des applications web robustes et sécurisées. Grâce à son architecture MVT et son admin intégré, je peux créer rapidement des applications complexes tout en maintenant un code propre et organisé.

React

Bibliothèque JavaScript moderne que j'utilise pour créer des interfaces utilisateur dynamiques et interactives. React me permet de développer des applications web performantes avec une expérience utilisateur fluide grâce à son système de composants réutilisables.

Flutter

Framework de développement mobile que j'utilise pour créer des applications multiplateformes natives. Avec Flutter, je peux développer une seule application qui fonctionne parfaitement sur iOS et Android tout en offrant des performances natives.

Canva

Outil de design intuitif que j'utilise pour créer des maquettes, des visuels attrayants et des assets graphiques pour mes projets web. Il me permet de donner vie aux interfaces utilisateur avec des designs modernes et professionnels.

Docker

Technologie de conteneurisation que j'utilise pour déployer mes applications de manière cohérente et isolée. Docker me permet d'assurer que mes applications fonctionnent de manière identique dans tous les environnements.
